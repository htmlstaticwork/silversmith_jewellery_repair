# Luxury Silversmith & Jewelry Repair Website

A premium, minimalist, craftsmanship-focused portfolio website for a silversmith specializing in custom silver jewelry and expert repair services. 

## Features
- **Design:** Deep Charcoal & Metallic Silver color palette.
- **Themes:** Supports Light and Dark modes.
- **Interactions:** Image gallery with zoom, before/after slider effect.
- **Responsiveness:** Fully mobile responsive.
- **Integrations:** Contact form, Google Maps, and WhatsApp Quick Contact.

## How to Test
Simply drag `pages/index.html` into your web browser or use a live server extension (like VSCode Live Server) to view the fully styled, responsive website.
